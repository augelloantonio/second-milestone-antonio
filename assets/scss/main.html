Welcome to #HelpYourBrain Challenge

This easy memory game is what you need to keep your mind sharp.

The goal of this game is to match the Web Logos! So when you click on one card it will show you a Logo, you have to match the same Logo with less number of attempt possible.

Instructions

  • Press the start button, the timer will start and you can start the game.

  • Click on one of the cards and see the image.

  • Click another card, if the card has the same image then is a Match, and you score a point!

  • If the images are not the same they will both flip back.
